## react설치

npx create-react-app .

## 필요한 라이브러리

npm i react-router-dom
npm i sass
npm i react-icons
npm i react-spinners

npm install react-modal

npm install axios
npm install react-helmet-async
npm install swiper
npm install --save-dev @babel/plugin-proposal-private-property-in-object

## build 배포

npm run build
